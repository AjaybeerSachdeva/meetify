# Conference Meeting Application

A React Native application built with **Expo** that allows users to book meeting rooms, view their reservations, and manage schedules efficiently.

## Features
- ✅ Book meeting rooms with real-time availability
- ✅ View and manage your bookings
- ✅ Prevent double bookings with room locking
- ✅ Responsive UI for mobile devices
- ✅ Built using Expo for easy development and deployment

## Tech Stack
- **React Native** (Expo)
- **Redux** for state management
- **JavaScript**
- **SQLite** for database creation and management

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AjaybeerSachdeva/meetify.git
   ```
2. Navigate to the project folder

3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npx expo start
   ```

## Usage
- Scan the QR code in Expo to run the app on your device.
- Use the app to book rooms and manage your meetings.
